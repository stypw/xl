通用卡片容器。

## 何时使用

最基础的卡片容器，可承载文字、列表、图片、段落，常用于后台概览页面。

```ts
import { XlAnchorModule } from '@stypw/xl';
```

## API

```html
<xlAnchorGroup>
    <xlAnchor nzTitle="卡片标题">卡片内容</xlAnchor>
</xlAnchorGroup>

```
